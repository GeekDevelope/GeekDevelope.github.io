Markdown 基本语法及 Markdown 文档模板

发表于 2021-12-14 更新于 2021-12-15

Markdown 是一种轻量级标记语言，排版语法简洁，让人们更多地关注内容本身而非排版。它使用易读易写的纯文本格式编写文档，可与 HTML 混编，可导出 HTML、PDF 以及本身的 .md 格式的文件。因简洁、高效、易读、易写，Markdown 被大量使用，如 Github、Wikipedia、简书等。

第3章 列表

3.1 无序列表

3.1.1 无序列表的使用，在符号 - 后加空格使用。如下：

  • 无序列表 1
  • 无序列表 2
  • 无序列表 3

3.1.2 如果要控制列表的层级，则需要在符号 - 前使用空格。如下：

  • 无序列表 1
  • 无序列表 2
    • 无序列表 2.1
    • 无序列表 2.2

3.2 有序列表

有序列表的使用，在数字及符号 . 后加空格后输入内容，如下：

  1. 有序列表 1
  2. 有序列表 2
  3. 有序列表 3

第4章 引用

引用的格式是在符号 > 后面书写文字。如下：

科学是一种强大的智慧力量，它致力于破除禁锢着我的神秘的桎梏。——高尔基

一面科技，一面艺术。——青年报 - 昨日
